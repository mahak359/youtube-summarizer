# youtube-summarizer

# 🎥 YouTube Summarizer using Gemini AI

## 🚀 Overview
This project converts YouTube videos into:
- 📄 Summaries
- 📝 Structured articles
- 🌐 Webpages (HTML, CSS, JS)

## 🧠 Technologies Used
- Python 3.11
- Google Gemini API
- YouTube Transcript API

## ⚙️ Workflow
YouTube Video → Transcript → Summary → Webpage → ZIP

## ▶️ How to Run
1. Install dependencies:
   pip install google-genai youtube-transcript-api python-dotenv

2. Create `.env` file:
   GOOGLE_API_KEY=your_api_key

3. Run:
   python open.py

## 📦 Output
- index.html
- style.css
- script.js
- website.zip

## 💡 Features
- Automatic transcript extraction
- AI-based summarization
- Webpage generation
- Easy to use
